# Deconvolution Project
This repository contains two distinct codebases:

## Deconvolution_Main_Code
This folder includes scripts for:
- Deconvoluting bulk RNA-seq data
- Running differential gene expression analysis to obtain cell type–corrected data
- Performing pseudobulk analysis applied to snRNA-seq data

## Deconvolution_Downstream_Analyses
This folder contains all downstream analyses applied to the differentially expressed genes. 
A more detailed description of the methods and workflows is provided inside this folder.
