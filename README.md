# Deconvolution Project
This repository contains two sets of code- Deconvolution_Main_Code and Deconvolution_Downstream_Analyses

# Deconvolution_Main_Code
The first set Deconvolution_Main_Code contains code for deconvoluting bulk RNA- seq data and consequently run differential gene expression in order to obtain cell type corrected data.
It also contains code pseudobulk analysis applied on snRNA-seq data

# Deconvolution_Downstream_Analyses
All the codes applied on Differentially expressed genes are contained in this folder. 
The detailed description is also inside this folder
